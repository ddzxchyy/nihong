# 长音（ちょうおん）

日语的音节有短音节（たんおん）和长音节（ちょうお）之分。



|  一般规则  |    词例    |     特殊场合     |
| :--------: | :--------: | :--------------: |
|  あ 段加   | おかあさん |        /         |
|  い段加い  |  にいさん  |        /         |
|  う段加う  |  つうやく  |                  |
|  え段加い  |  せんせい  | 加 え ：ねえさん |
|  お段加う  | おとうさん | 加 お：おおきい  |
| 外来语加 - |   のーと   |        /         |
