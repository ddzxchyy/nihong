# 长音（ちょうおん）

日语的音节有短音节（たんおん）和长音节（ちょうお）之分。



|  一般规则   |    词例    |     特殊场合     |
| :---------: | :--------: | :--------------: |
| あ 段加 あ  | おかあさん |        /         |
| い 段加 い  |  にいさん  |        /         |
| う 段加 う  |  つうやく  |        /         |
| え 段加  い |  せんせい  | 加 え ：ねえさん |
| お 段加 う  | おとうさん | 加 お：おおきい  |
| 外来语加 -  |   ノート   |        /         |
